Instructions for use of Receiptify:
1. Compile the program by navigating to the directory with all code in it and running "javac *.java" from the command line
2. Run the program with "java main"
3. To use the program, click on the "Input Image" button and navigate through your file directory to the desired receipt image in JPG or JPEG format.
5. Then, select the area you would like to crop by clicking in one corner of the desired area, holding and dragging your cursor to the opposite corner, then releasing.
6. This selection will then be output and saved as a jpg file.s
